TO RUN THE CODE, RUN THIS IN THE TERMINAL:
"cd 'c:\Users\yourUserHere\...\AppJavaFX-main\AppJavaFX-main'; mvn clean javafx:run"
^
i.e. cd 'c:\Users\myUser\Downloads\AppJavaFX-app\AppJavaFX-main'; mvn clean javafx:run

Once this is ran at least once it should work for the rest of the session by only using the cmd
"mvn clean javafx:run"

KEYBINDS:
R - Rectangle
T - Circle
Y - Clear Canvas

To read the tooltip on a shape, click on an entry of a shape you drew on the list to the right of the screen.
To swap shapes, press R/T or click the buttons in the top left of the screen.
